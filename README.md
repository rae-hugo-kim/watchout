# watchout

갤럭시 워치8 초보(부모님)용 따라하기 학습 페이지.

## 로컬 실행

```bash
cd watchout
python3 -m http.server 4173
```

브라우저에서 아래 주소 열기:

<http://localhost:4173>

## 파일 구조
- `PLAN.md` : 작업 계획 문서
- `index.html` : 메인 페이지
- `styles.css` : 스타일
- `script.js` : 체크리스트/인터랙션
- `assets/*.svg` : 안내용 이미지
