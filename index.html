빠른 시작 (처음 5분)

  1. 워치를 손목에 착용하고 전원을 켜세요.
  2. 휴대폰과 블루투스 연결이 되었는지 확인하세요.
  3. 아래 4개를 순서대로 해보세요.
기본 조작 전화 받기 내 핸드폰 찾기 건강 측정/운동 시작

1) 기본 조작

목표: 화면 켜기, 스와이프, 버튼 사용 익히기

스와이프(쓸어넘기기): 손가락을 화면에 댄 채로 한 방향으로 미는 동작이에요.

  1. 워치 화면을 한번 눌러 켭니다.
  2. 왼쪽/오른쪽으로 쓸어넘겨 위젯을 넘겨봅니다.
  3. 위에서 아래로 쓸어내려 빠른 설정을 엽니다.
  4. 아래에서 위로 쓸어올려 앱 목록을 엽니다.
  5. 옆 버튼을 눌러 홈으로 돌아갑니다.

2) 전화 받기

목표: 전화 오면 워치에서 바로 받기

  1. 전화가 오면 화면의 초록 버튼을 누릅니다.
  2. 말할 때는 워치를 입 가까이 가져갑니다.
  3. 통화를 끝낼 때는 빨간 버튼을 누릅니다.
  4. 소리가 크면 측면 버튼으로 볼륨을 조절합니다.

3) 내 핸드폰 찾기

목표: 휴대폰이 안 보일 때 벨 울리기

  1. 워치 화면을 위에서 아래로 내려 빠른 설정을 엽니다.
  2. "휴대폰 찾기" 아이콘을 누릅니다.
  3. 휴대폰에서 소리가 나면 소리를 따라 찾습니다.
  4. 찾았으면 휴대폰 화면에서 소리 끄기를 누릅니다.

4) 건강 측정 + 운동 시작

목표: 건강 앱에서 기본 측정 3가지 + 운동 시작까지 해보기

  1. 앱 목록에서 Samsung Health를 엽니다.
  2. 심박수 측정을 누르고 30초간 가만히 있습니다.
  3. 스트레스 측정을 누르고 편하게 호흡합니다.
  4. 체성분 측정은 안내 자세를 따라 15초 유지합니다.
  5. 운동 시작: Samsung Health에서 운동을 누릅니다.
  6. 걷기 또는 러닝을 선택하고 시작 버튼을 누릅니다.
  7. 끝낼 때는 화면을 밀어 종료를 누르고 기록을 저장합니다.

주의: 측정 전 손목을 건조하게 하고, 밴드를 너무 느슨하지 않게 착용하세요.

다음 단계로 가기

진행률: 0 / 4 완료

심화 과정 1) 제미나이 호출해서 물어보기

목표: 궁금한 걸 말로 바로 물어보고 답 듣기

  1. 워치에서 앱 목록을 열고 Gemini를 실행합니다.
  2. 마이크 버튼을 누른 뒤 또박또박 질문합니다.
  3. 예시: "오늘 날씨 알려줘", "10분 뒤 알람 맞춰줘"
  4. 답변이 길면 "짧게 다시 말해줘"라고 다시 요청합니다.

※ 워치에서 Gemini가 바로 안 보이면, 휴대폰 Galaxy Wearable 앱에서 앱 동기화를 먼저 확인하세요.

심화 과정 2) 음성으로 일정/알람 만들기

  1. Gemini에 "내일 오전 9시에 병원 알림"이라고 말합니다.
  2. 등록 내용이 맞는지 확인 후 저장합니다.
  3. 같은 방식으로 약 복용/산책/운동 알림을 만듭니다.

심화 과정 3) 위급 시 빠른 연락 연습

  1. 연락처 즐겨찾기에 가족 번호를 등록합니다.
  2. 워치에서 최근 통화 또는 즐겨찾기에서 바로 전화합니다.
  3. 실전처럼 1번 연습해두면 실제 상황에서 훨씬 빨라집니다.
